ASYMPTOTIC NOTATION SUMMARY
==========================
1. Linear Search by ID:       O(n) - compares up to n elements
2. Binary Search by ID:       O(log n) - divides search space by 2
3. Hash Table Search by ID:   O(1) - direct access via hash map
4. Linear Search by Name:     O(n) - checks all products
5. Range Search:              O(n) - scans entire list

PERFORMANCE COMPARISON (for n = 10):
- Linear Search: up to 10 operations
- Binary Search: up to 4 operations
- Hash Table Search: 1 operation

KEY TAKEAWAYS:
✓ Choose appropriate data structures based on access patterns
✓ Hash tables offer O(1) search for key-based lookups
✓ Binary search requires sorted data but improves from O(n) to O(log n)
✓ Understand trade-offs between space and time complexity
